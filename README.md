# 💫 About Me
I'm a Flutter / Mobile Engineer focused on designing, building, and shipping production-grade cross-platform apps to the App Store and Google Play. I work day-to-day in Dart and Flutter with Provider, Riverpod, and Bloc for state management, Firebase for backend services, and native platform integration for things like biometrics, camera/OCR, and secure on-device storage.<br><br>I've shipped live, multilingual apps in regulated, high-stakes domains — including <a href="https://play.google.com/store/apps/details?id=com.lof.theracare">TheraCare AI</a>, a healthcare app on both stores — with a strong focus on mobile security, on-device data protection, performance, and accessibility. Alongside the mobile work, I bring a solid backend and AI foundation, integrating LLMs, on-device OCR, and real-time data into the apps I build.<br><br>I care about clean architecture, polished UX, and shipping things that actually hold up in production. Always learning, always iterating.
